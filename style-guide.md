# 新能源汽车日报 - 样式指南文档

## 设计系统概述

本样式指南基于Apple设计美学，采用极简主义设计原则，建立了完整的设计系统和组件库。

## 设计原则

### 1. 极简主义
- 去除所有非必要的视觉元素
- 保持界面干净整洁
- 强调内容本身而非装饰

### 2. 清晰层次
- 建立严格的视觉层级
- 使用字号、字重、颜色区分重要性
- 保持一致的间距和布局

### 3. 优雅交互
- 平滑的过渡动画
- 微妙的状态反馈
- 直观的用户引导

## 设计规范

### 颜色系统
```css
--primary-color: #1d1d1f;      /* 主文本颜色 */
--secondary-color: #86868b;    /* 次要文本颜色 */
--accent-color: #0071e3;       /* 强调色 */
--background-color: #fbfbfd;   /* 背景色 */
--card-background: #ffffff;    /* 卡片背景 */
--border-color: #d2d2d7;       /* 边框色 */
```

### 字体系统
- **主字体**: SF Pro Display / SF Pro Text
- **字号层级**:
  - H1: 48px / 40px / 32px / 28px (响应式)
  - H2: 32px / 28px / 24px / 22px
  - H3: 24px / 22px / 20px
  - 正文: 17px / 16px / 15px
  - 小字: 14px

### 间距系统 (8pt网格)
```css
--grid-unit: 8px;
--space-xs: 8px;     /* 小间距 */
--space-sm: 16px;    /* 中间距 */
--space-md: 24px;    /* 中等间距 */
--space-lg: 32px;    /* 大间距 */
--space-xl: 48px;    /* 超大间距 */
```

### 圆角系统
```css
--border-radius: 12px;  /* 卡片圆角 */
--radius-sm: 6px;       /* 小圆角 */
--radius-lg: 16px;      /* 大圆角 */
```

### 阴影系统
```css
--shadow: 0 4px 16px rgba(0, 0, 0, 0.08);     /* 默认阴影 */
--shadow-hover: 0 8px 24px rgba(0, 0, 0, 0.12); /* 悬停阴影 */
```

## 组件库

### 1. 头部区域 (Header)
- **背景**: 渐变色彩 (#667eea → #764ba2)
- **图标**: 64px / 48px (响应式)
- **标题**: 48px / 32px / 28px
- **副标题**: 21px / 18px
- **内边距**: 80px 0 / 60px 0 / 40px 0

### 2. 内容卡片 (Section Card)
- **背景**: 白色 (#ffffff)
- **圆角**: 12px
- **阴影**: 0 4px 16px rgba(0, 0, 0, 0.08)
- **悬停**: 上移2px，阴影增强
- **内边距**: 40px / 32px / 24px (响应式)

### 3. 新闻条目 (News Item)
- **背景**: #f5f5f7 (浅灰色)
- **边框**: 左侧3px强调线
- **圆角**: 12px
- **悬停**: 背景变深，右移4px
- **间距**: 32px 下方间距

### 4. 数据表格 (Data Table)
- **表头背景**: #f5f5f7
- **表头文字**: 14px，大写，字母间距0.05em
- **表格行**: 悬停时 #fafafa
- **排名标记**: 金、银、铜色渐变

### 5. 统计卡片 (Stat Card)
- **数字**: 40px / 32px / 28px，字重600
- **标签**: 14px，大写，字母间距0.05em
- **顶部边框**: 3px强调色
- **悬停**: 上移4px，阴影增强

## 响应式设计

### 断点设置
- **桌面端**: > 1024px
- **平板端**: 768px - 1024px
- **手机端**: 480px - 768px
- **小屏手机**: < 480px

### 响应式规则
1. **字体缩放**: 按断点逐级缩小
2. **间距调整**: 内边距和间距相应减少
3. **布局变化**: 单列布局，网格重构
4. **组件适配**: 触摸友好的交互区域

## 交互设计

### 动画系统
- **过渡时间**: 0.3s
- **缓动函数**: cubic-bezier(0.4, 0, 0.2, 1)
- **加载动画**: 数字递增，透明度渐变
- **悬停效果**: 位移、阴影、颜色变化

### 状态反馈
- **链接**: 下划线出现，颜色加深
- **按钮**: 按压效果，颜色变化
- **卡片**: 悬停上浮，阴影增强

## 可访问性

### 颜色对比
- **主文本**: 7:1 对比度
- **次要文本**: 4.5:1 对比度
- **链接色**: 3:1 对比度

### 字体优化
- **抗锯齿**: -webkit-font-smoothing
- **字重**: 避免过细或过粗
- **行高**: 1.6 倍，确保可读性

### 键盘导航
- **焦点指示**: 清晰的焦点样式
- **跳过链接**: 提供内容跳转
- **语义化**: 正确的HTML结构

## 性能优化
